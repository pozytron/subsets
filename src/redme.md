# task:
Find max value of sub set of 3 values from a list of integers.

input: 
`List<Integer> values;`

return: 
`[ x, y ]`

where:

`x - sum value and`

`y - subset index`

Given data: `[1,2,5,87,1]`

> 1+2+5 = 8         (subset 0)
> 
> 2+5+87 = 94       (subset 1)
> 
> 5+85+1 = 91       (subset 2)

### example:
```javascript
//eg:
values = [1,2,5,87,1, ...more]

//return: 
[94, 1] 
```
